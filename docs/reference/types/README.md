# Types

PyneCore's type system and how it differs from Pine Script.

## Key Difference

Pine Script uses four type qualifiers (`const`, `input`, `simple`, `series`) that the
programmer must track. PyneCore eliminates this — a value is simply a value, and Series
behavior is created automatically when needed.
